# Alt Text Generator KDR

Automatically generate SEO-friendly alt text for WordPress images.

## Features
- Bulk generate alt text
- AI-powered image description
- Media library integration
- Lightweight & fast

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the WordPress admin panel
3. Configure from Media → Alt Text Generator KDR

## Requirements
- WordPress 6.0+
- PHP 7.4+
